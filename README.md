## Project Overview
This is a **Music Player Web App** built with **React.js** and **Tailwind CSS**.  
Users can:
- Play / Pause music
- Skip Next / Previous songs
- Click on any song card to play
- See a persistent footer music player

---

## Tech Stack
- **React.js** - UI Library
- **Tailwind CSS** - Styling
- **Lucide React** - Icons
- **React Router DOM** - Routing (if used)
- **Swiper.js** - Carousel / Slider (if used)

---

## Features
- Persistent footer player (always visible)
- Play / Pause / Next / Prev controls
- Click on any song card to play
- Progress bar with click-to-seek
- Responsive layout (mobile & desktop)
- Fixed footer layout to prevent shifting icons
- Truncated long song titles / artists
